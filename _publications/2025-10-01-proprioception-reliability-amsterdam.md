**Background**

Conventional proprioception assessments are subjective, ordinal, and insensitive to small changes, limiting their use in longitudinal research. Robotic assessment offers finer resolution, but psychometric data for wrist-focused proprioceptive batteries spanning multiple reference frames are limited.

**Methods**

Twenty neurotypical adults completed two identical sessions on the same day (≥2 hours apart) using PLUTO, a single-degree-of-freedom wrist robot. Three tasks were evaluated:

- **JDT** — passive movement detection threshold (single frame)
- **J-to-J** — joint-to-joint angle matching (same frame)
- **J-to-V** — joint-to-visual matching, requiring coordinate transformation across reference frames (cross-frame)

Relative reliability was quantified with ICC(2,1); absolute reliability with SEM and SDC₉₅.

**Results**

| Task | ICC(2,1) [95% CI] | SDC₉₅ |
|------|-------------------|--------|
| JDT | 0.959 [0.900–0.980] | 0.85° |
| J-to-J | 0.837 [0.640–0.930] | 1.71° |
| J-to-V | 0.769 [0.500–0.900] | 3.54° |

Error scaled monotonically across tasks (JDT < J-to-J < J-to-V; all p < 0.01). No learning effect observed.

**Conclusions**

All three tasks showed good-to-excellent within-day reliability. SDC₉₅ values provide task-specific thresholds for detecting true change in future intervention trials. The error hierarchy is consistent with increasing computational demands across proprioceptive reference frames.
